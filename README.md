## Getting Started

_Berikut ini adalah cara mengakses aplikasi._

1. Clone repository
2. Buka pada Android Studio
3. Pastikan Versi Android Studio dengan Aplikasi sesuai (Minimal Android Studio Hedgehog)
5. Run aplikasi
